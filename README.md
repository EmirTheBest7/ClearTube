## 
A lightweight and efficient browser extension that blocks ads on YouTube, providing a clean and uninterrupted viewing experience. Built with performance and simplicity in mind, it removes video ads, banners, and pop-ups without slowing down playback.

## 🛠️ Installation Guide

### 🌐 Chrome, Brave, Edge, and Opera
1.  **Download:** Get the latest `CleanTube.zip` from the [Releases](https://github.com/EmirTheBest7/CleanTube/releases) section.
2.  **Extract:** Unzip the folder to a safe location on your drive.
3.  **Extensions Page:** Open your browser and go to `chrome://extensions/`.
4.  **Developer Mode:** Toggle **"Developer mode"** (top right) to **ON**.
5.  **Load:** Click **"Load unpacked"** and select the folder containing the `manifest.json`.

### 🦊 Mozilla Firefox
1.  **Access Debugging:** Type `about:debugging` in your address bar and press Enter.
2.  **This Firefox:** Click **"This Firefox"** in the left menu.
3.  **Load Add-on:** Click **"Load Temporary Add-on..."** and select the `manifest.json` file from your extracted folder.
    * *Note: Temporary extensions are removed when Firefox restarts.*

## 🤝 Community & Contributions

CleanTube is an open-source project built for the community. We encourage developers to add their own algorithms to our library to stay ahead of intrusive advertising. See `CONTRIBUTING` for more information.

### How to contribute:
1.  **Fork** the repository.
2.  Add your logic to the `_src` directory.
3.  Include a brief description of your changes.
4.  Submit a **Pull Request**.

> [!IMPORTANT]
